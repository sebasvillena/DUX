How to play

If the game mode selected is "drag images", you have to select the elements you think belong to an emergency bag, and drag them into the picture of the bag.

If the game mode selected is "select item", you have to select the element you consider belong to an emergency bag, and then press the button "Select option".


The game mode can be changed in the configuration menu. You can access such menu from the main screen, or from the pause menu in the game.


The objective of the game is learn what things can be of help in case of an emergency, while also learning why they are useful (or not).

Game introduction

One can never tell when a catastrophe could occur, thats why it's good to always be prepared for them.
In this game you will learn what elements are important to have in a bag in case of emergencies, and the reason why each of them are important.

To play you simply must select the elements you think belong to an emergency bag, and then drag them onto the picture of the bag, or press the button "Select option", depending on the game mode selected on the configuration menu.

If you need to, you can access the configurations from the pause menu.
